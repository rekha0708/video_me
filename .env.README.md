# .env Configuration Quick Reference

## 🎯 Quick Setup (3 steps)

```bash
# 1. Copy template
cp .env.example .env

# 2. Add your HuggingFace token
nano .env
# Find: HF_TOKEN=hf_REPLACE_WITH_YOUR_TOKEN_HERE
# Replace with your actual token from https://huggingface.co/settings/tokens

# 3. Load and verify
source .env
echo $HF_TOKEN  # Should show: hf_...
```

**Done!** Now run: `bash scripts/setup_gpu.sh`

## ⚙️ Common Customizations

### **Change Language**
```bash
VIDEO_ME_TARGET_LANGUAGE=hi    # Hindi
VIDEO_ME_TARGET_LANGUAGE=both  # English + Hindi
```

### **CPU-only Testing**
```bash
VIDEO_ME_WHISPER_DEVICE=cpu
VIDEO_ME_WHISPER_COMPUTE_TYPE=int8
```

### **Skip Human Approvals (CI/testing)**
```bash
VIDEO_ME_AUTO_APPROVE_PLAN=true
VIDEO_ME_AUTO_APPROVE_IMAGES=true
```

### **Use PostgreSQL + S3 (production)**
```bash
VIDEO_ME_JOB_STORE=postgres
VIDEO_ME_ARTIFACT_STORE=s3
VIDEO_ME_POSTGRES_DSN=postgresql://user:pass@host:5432/video_me
VIDEO_ME_S3_ENDPOINT_URL=https://s3.amazonaws.com
VIDEO_ME_S3_BUCKET=your-bucket
VIDEO_ME_S3_ACCESS_KEY_ID=your-key
VIDEO_ME_S3_SECRET_ACCESS_KEY=your-secret
```

## 📝 Full Variable List

### **HuggingFace**
- `HF_TOKEN` — Token for downloading gated models (Flux 2.0)

### **Models**
- `VIDEO_ME_LLM_MODEL` — LLM for text generation (default: `qwen3.6:35b`)
- `VIDEO_ME_CRITIQUE_MODEL` — VLM for image/video critique (default: `qwen3.6:35b`)
- `VIDEO_ME_WHISPER_MODEL_SIZE` — Transcription quality (default: `medium`)

### **Adapters**
- `VIDEO_ME_RENDER_ADAPTER` — Image generation (default: `comfyui_flux`)
- `VIDEO_ME_VIDEO_ADAPTER` — Video generation (default: `ltx`)
- `VIDEO_ME_TTS_ADAPTER` — Text-to-speech (default: `fish_s2`)

### **Service URLs**
- `VIDEO_ME_COMFYUI_BASE_URL` — ComfyUI (default: `http://localhost:8188`)
- `VIDEO_ME_FISH_S2_BASE_URL` — Fish Audio S2 (default: `http://localhost:8025`)
- `VIDEO_ME_LLM_BASE_URL` — Ollama (default: `http://localhost:11434/v1`)

### **Directories**
- `VIDEO_ME_DATA_DIR` — Job working directories (default: `.local`)
- `VIDEO_ME_REVIEW_DIR` — Published videos (default: `review`)
- `VIDEO_ME_LORA_DIR` — LoRA weights (default: `loras`)
- `VIDEO_ME_VOICE_DIR` — Voice reference files (default: `voices`)

### **Approval Gates**
- `VIDEO_ME_AUTO_APPROVE_PLAN` — Skip plan approval (default: `false`)
- `VIDEO_ME_AUTO_APPROVE_IMAGES` — Skip image approval (default: `false`)
- `VIDEO_ME_APPROVAL_PORT` — Approval UI port (default: `8765`)
- `VIDEO_ME_MAX_PLAN_ITERATIONS` — Max critique re-plans (default: `3`)

### **Image Critique**
- `VIDEO_ME_IMAGE_CANDIDATES` — Images per shot (default: `3`)
- `VIDEO_ME_FEEDBACK_LOG_DIR` — Self-learning log (default: `assets/kids_duo`)

### **Storage**
- `VIDEO_ME_JOB_STORE` — Job metadata (`sqlite` or `postgres`)
- `VIDEO_ME_ARTIFACT_STORE` — File storage (`local` or `s3`)
- `VIDEO_ME_POSTGRES_DSN` — PostgreSQL connection string
- `VIDEO_ME_S3_*` — S3/MinIO configuration

### **GPU Settings**
- `VIDEO_ME_WHISPER_DEVICE` — Transcription device (`cpu` or `cuda`)
- `VIDEO_ME_WHISPER_COMPUTE_TYPE` — Precision (`int8` or `float16`)

### **Language**
- `VIDEO_ME_TARGET_LANGUAGE` — Output language (`en`, `hi`, or `both`)

## 🔒 Security Checklist

- ✅ `.env` is in `.gitignore` (never commit)
- ✅ Use read-only HuggingFace tokens
- ✅ Set token expiration (30-90 days)
- ✅ Don't share `.env` file
- ✅ Rotate tokens regularly

## 🆘 Troubleshooting

**Problem:** `echo $HF_TOKEN` shows nothing

**Solution:**
```bash
source .env
echo $HF_TOKEN
```

**Problem:** Download fails with "403 Forbidden"

**Solution:** Token needs gated repo access. Check at https://huggingface.co/settings/tokens

**Problem:** "You need to accept the license"

**Solution:** Visit https://huggingface.co/black-forest-labs/FLUX.2-dev and click "Agree"
